# HABIBTI Frontend

React web application for HABIBTI - Privacy-first chat app.

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Axios

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Copy `.env.example` to `.env` and configure:

```
VITE_API_URL=http://localhost:8000/api/v1
```
